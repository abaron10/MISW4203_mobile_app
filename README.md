#### Para correr la aplicación:
1. Clonar el repositorio https://github.com/abaron10/MISW4203_mobile_app.git
2. Entrar a la app con el editor de texto [Android Studio](https://developer.android.com/studio?hl=es-419&gclid=Cj0KCQjw8p2MBhCiARIsADDUFVGOozYhPWmZoddmeyVkkqhtGqGZoZeVlEif5gwLh3C-1cOWOzP-VKwaAg_JEALw_wcB&gclsrc=aw.ds).
3. Al entrar al editor de texto se debe seleccionar un dispositivo asociado al emulador, seleccionar el que mas se adecue al sistema operativo , en este caso el Pixel 3.
<p align="center">
<img src="https://user-images.githubusercontent.com/64280930/140660981-653f8ab8-b121-4099-a4e4-bc7ae1c1110b.png">
</p>
4. Posteriormente y a modo de recompilar la app clonada , ir a la sección de Build y dar click en Make Proyect
<p align="center">
<img src="https://user-images.githubusercontent.com/64280930/140661018-23bf2b93-5d32-4909-94ab-d3e3f93a9515.png">
</p>
5. Una vez el build sea exitoso se puede correr la app, con el botón de play se inicia el proyecto.

#### Navegando la aplicación
1. Una vez ejecutado el botón de play , la app le permitira logearse como usuario o como coleccionista. En este caso navegaremos la vista de usuario, Dar click en este botón.
<p align="center">
<img src="https://user-images.githubusercontent.com/64280930/140661131-f8e18263-6df7-4cf7-869f-dca0bbe68377.png" width="400" height="790">
</p>
2. En el menú de navegación lateral encontrara las siguientes opciones , ir a albumes.
<p align="center">

</p>
